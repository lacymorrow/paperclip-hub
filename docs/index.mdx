# <SiteName /> Documentation

Paperclip Hub is the plugin directory for [Paperclip](https://paperclip.ing) — a curated catalog of connectors, providers, tools, and memory backends, published by the community. Browse the registry, find a plugin, and wire it into your agent.

These docs cover **using the hub**: browsing and searching plugins, submitting your own, and the registry format. For Paperclip itself — the CLI, the plugin manifest, and installing a plugin — see [paperclip.ing](https://paperclip.ing).

## Use the hub

| Task                | Go to                                        |
| ------------------- | -------------------------------------------- |
| Browse every plugin | [Home](/)                                    |
| Filter by category  | [`/?category=provider`](/?category=provider) |
| Sort by newest      | [`/?sort=newest`](/?sort=newest)             |
| Submit a plugin     | [Submit](/submit)                            |

## Categories

Every plugin declares one `category`. The browse UI filters on these (defined in [`registry/schema.json`](https://github.com/lacymorrow/paperclip-hub/blob/main/registry/schema.json)):

| Category        | For                                 |
| --------------- | ----------------------------------- |
| `auth`          | Authentication providers            |
| `provider`      | Model and service providers         |
| `tools`         | Tools the agent can call            |
| `integration`   | Third-party integrations            |
| `observability` | Logging, tracing, and analytics     |
| `memory`        | Memory and storage backends         |
| `other`         | Anything that doesn't fit the above |

## Submit a plugin

Share your work from the [Submit page](/submit): fill in the form and it opens a pre-filled pull request on GitHub that adds your entry to the registry — no hub account required. Prefer the terminal? `paperclip plugin submit` opens the same PR.

## The registry

The catalog is a set of JSON files under `registry/plugins/`, each validated against [`registry/schema.json`](https://github.com/lacymorrow/paperclip-hub/blob/main/registry/schema.json). A submission adds one file there via pull request.
