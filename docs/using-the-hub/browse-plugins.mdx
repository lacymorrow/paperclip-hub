# Browse Plugins

The [hub home](/) lists every plugin in the registry. Use the search box to find by keyword, filter by [category](/docs/registry/categories) (for example [`/?category=provider`](/?category=provider)), or show the newest first with [`/?sort=newest`](/?sort=newest).

Each result links to a plugin's detail page — what it does, the capabilities it implements, its source repository, and its install command. This page will expand with tips on evaluating and comparing plugins before you install.
