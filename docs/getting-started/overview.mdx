# Overview

Paperclip Hub is the community directory of plugins for [Paperclip](https://paperclip.ing) — where you find connectors, providers, tools, and memory backends that extend an agent, and where authors publish their own.

These docs are organized by audience. **Getting Started** and **Using the Hub** are for anyone installing plugins; **Plugin Authors** covers publishing your own; and **Registry** and **Reference** document the underlying data and answer common questions. New here? Jump straight to the [Quickstart](/docs/getting-started/quickstart).
