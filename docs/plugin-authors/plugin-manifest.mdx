# Plugin Manifest

A Paperclip plugin is an npm package that exposes a manifest describing its capabilities and entry points. The manifest is what Paperclip reads to load and run your plugin.

The manifest format is owned by Paperclip itself — see [paperclip.ing](https://paperclip.ing) for the authoritative spec. This page will summarize the fields that matter for getting listed in the hub and link out for the full reference.
