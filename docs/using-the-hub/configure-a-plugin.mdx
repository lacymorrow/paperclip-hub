# Configure a Plugin

Most plugins need some configuration — API keys, options, or environment variables — before they're useful. Configuration is specific to each plugin and documented in its own repository.

This page will outline the general pattern for configuring a plugin in Paperclip and link out to each plugin's own docs for specifics. The hub lists plugins; [Paperclip](https://paperclip.ing) runs and configures them.
