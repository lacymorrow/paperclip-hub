Langfuse Tracing

by langfuse
observability verified

Full observability for your AI agents with distributed tracing, cost tracking, and prompt management. Automatically instruments all LLM calls, tool executions, and agent lifecycle events.

Installation

$ paperclip plugin install paperclip-otel

Capabilities

event config chat.message chat.params tool.execute.before tool.execute.after

Getting Started

After installation, configure your Langfuse credentials in your agent config:

# .env
LANGFUSE_PUBLIC_KEY=pk-lf-...
LANGFUSE_SECRET_KEY=sk-lf-...
LANGFUSE_HOST=https://cloud.langfuse.com

Features

  • Automatic tracing of all LLM calls with latency, tokens, and cost
  • Tool execution tracking with input/output capture
  • Session-based grouping for multi-turn conversations
  • Custom metadata and tags for filtering traces
  • Real-time cost dashboard with per-model breakdowns

Configuration

// paperclip.config.ts
{
  plugins: [
    ["paperclip-otel", {
      traceAllTools: true,
      sessionTracking: true,
      costTracking: true
    }]
  ]
}
