# Getting Started with Paperclip Plugins

Paperclip plugins extend the AI agent platform with new capabilities — tools, connectors, auth providers, event hooks, and more. This guide walks you through building your first plugin and submitting it to the Hub.

## What is a Paperclip Plugin?

A plugin is an npm package that implements one or more Paperclip capability hooks:

- **tool** — expose functions the agent can call
- **event** — react to agent lifecycle events
- **auth** — add authentication providers
- **connector** — integrate external services
- **chat.message** / **chat.params** — shape conversation flow

## Step 1: Scaffold Your Plugin

```bash
mkdir my-paperclip-plugin
cd my-paperclip-plugin
bun init -y
```

Install the Paperclip SDK:

```bash
bun add @paperclip/sdk
```

## Step 2: Implement a Capability

Here's a minimal tool plugin:

```typescript
// index.ts
import { defineTool } from "@paperclip/sdk";

export default defineTool({
  name: "fetch-weather",
  description: "Fetch current weather for a location",
  parameters: {
    location: { type: "string", description: "City or coordinates" },
  },
  async execute({ location }) {
    // your implementation here
    return { temperature: "72°F", condition: "Sunny", location };
  },
});
```

## Step 3: Publish to npm

```bash
# Make sure your package.json has a valid name and version
npm publish --access public
```

Your package name becomes the `npmPackage` field in the registry.

## Step 4: Submit to the Hub

Once published, submit your plugin at [cliphub.fyi/submit](https://cliphub.fyi/submit):

1. Fill out the form with your plugin name, npm package, description, and capabilities
2. Click **Open pull request on GitHub**
3. GitHub pre-fills the registry file — click **Commit** to open a PR
4. The Paperclip team reviews and merges

No account required on the Hub — GitHub handles attribution natively.

## Plugin Registry Format

Each plugin in the registry is a JSON file at `registry/plugins/<package-name>.json`:

```json
{
  "$schema": "../schema.json",
  "name": "My Plugin",
  "npmPackage": "my-paperclip-plugin",
  "description": "What your plugin does.",
  "category": "tools",
  "capabilities": ["tool"],
  "author": "your-github-username",
  "submittedAt": "2025-05-01T00:00:00.000Z"
}
```
