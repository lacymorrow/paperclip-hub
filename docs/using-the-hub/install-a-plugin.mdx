# Install a Plugin

Every plugin's detail page has an **Install** panel with the exact command for that plugin — copy it and run it in your project. You'll need [Paperclip](https://paperclip.ing) itself installed first.

This page will cover installation in more detail, including prerequisites and how to confirm a plugin loaded correctly. For the quickest path end to end, see the [Quickstart](/docs/getting-started/quickstart).
