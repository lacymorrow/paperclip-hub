# Versioning

A plugin's version comes from its npm package — the registry entry references the package rather than pinning a version, so updates flow from npm.

This page will document how versions surface on plugin pages and any conventions authors should follow (for example semantic versioning). For the fields a registry entry stores today, see the [Registry Schema](/docs/registry/schema).
