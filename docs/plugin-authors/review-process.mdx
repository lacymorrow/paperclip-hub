# Review Process

Submitting a plugin opens a pull request that adds your entry to the registry. A maintainer reviews it for a valid schema, an appropriate category and capabilities, and a working package before merging.

This page will document what reviewers look for, typical turnaround, and how to respond to change requests. To start a submission, see [Submit a Plugin](/docs/plugin-authors/submit-a-plugin).
